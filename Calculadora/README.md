# Calculadora

Calculadora de quatro operações desenvolvida em equipe como parte da Sprint 1 de ENG4021.

## Arquivos

| Arquivo | Responsável | Função |
|---|---|---|
| `calculadora.py` | Aluno 1 | Programa principal |
| `soma.py` | Aluno 2 | Soma (`somaf`) |
| `subtrai.py` | Aluno 3 | Subtração (`subtraif`) |
| `multiplica.py` | Aluno 4 | Multiplicação (`multiplicaf`) |
| `divide.py` | Aluno 5 | Divisão (`dividef`) |

## Como executar

Dentro da pasta `Calculadora`, rode:

```bash
python calculadora.py
```

Digite dois números e um operador (`+`, `-`, `*`, `/`) quando solicitado.
